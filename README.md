# 卡牌对战 · MVP

纯前端（HTML + CSS + ES Module）的双人对战卡牌游戏。无构建步骤、无依赖打包，**直接打开就能玩**。

---

## 快速开始

### 方式 1：直接双击（最快）

双击 `index.html`，用浏览器打开即可。

> ⚠️ 部分浏览器对 `file://` 协议下的 ES Module 有限制。如果界面没出来或控制台报 CORS 错，用方式 2。

### 方式 2：本地服务器（推荐）

任意一个静态服务器都行，挑你顺手的：

```bash
# Python（系统自带或 python.org）
python -m http.server 8765

# Node（如果装了 npx）
npx serve .

# 或
npx http-server -p 8765
```

然后浏览器打开 <http://localhost:8765/>。

### 方式 3：在线玩

(可选) 部署到 GitHub Pages / Vercel / Netlify，整个目录拖上去即可。

## 玩法教程

### 胜利条件

把**对方主公的血量打到 0**（共 3 滴血）。己方主公血量归零则失败。

### 战场布局

双方各占一个 2 行 × 3 列的阵地，**镜像对称**：

```
       后排  [5]  [3]  [1]        ← 敌方阵地（在你对面）
             ─────────────
       前排  [6]  [4]  [2]
  ════════════════════════════
       前排  [2]  [4]  [6]
             ─────────────
       后排  [1]  [3]  [5]        ← 我方阵地
```

编号是「全局攻击序号」：**先手方先攻，按 1→2→3→4→5→6 交替**。每回合双方互换先手。

### 一回合的 5 个阶段

界面右下角会显示当前阶段：

| 阶段 | 你需要做什么 |
|---|---|
| **DRAW 抽牌** | 自动：双方各抽 3 张（首回合开局抽 7 张） |
| **DEPLOY 部署** | 点手牌 → 选要弃的代价牌 → 点空格放置（**默认暗置**）。点「完成部署 → 翻面」结束 |
| **REVEAL 亮牌** | 自动：双方同时翻开，触发盗贼孤立先手等亮牌效果 |
| **BATTLE 战斗** | 自动播放：按全局序号轮流攻击。**弓箭手/炸弹人/祭司/亡灵法师**轮到我时会高亮敌方可击目标，**点击选择** |
| **END 结束** | 点「结束回合 →」进入下一回合 |

### 出牌代价

每出 1 张单位牌，必须**额外弃 1 张手牌**作为代价（吸血战士要弃 2 张）。手牌只有 1 张时无法出牌。

变羊术（法术）免费释放。

### 单位速查

| 单位 | 血/攻 | 能力 |
|---|---|---|
| ⚔️ 剑士 | 2 / 1 | **穿刺**：攻击打目标一整列（前后排都吃伤害） |
| 🏹 弓箭手 | 2 / 1 | **任意射**：可选敌方任意位置 |
| 🛡️ 盾兵 | 2 / 1 | **替伤**：为上下左右友军挡伤害 |
| 🪓 斧兵 | 3 / 1 | 无能力（血厚肉盾） |
| 💣 炸弹人 | 1 / 1 | **先发**：仅在本回合部署时，战斗开始前对任意敌方位置打 1；之后按弓箭手规则攻击 |
| 🗡️ 盗贼 | 2 / 1 | **孤立先手**：亮牌时若上下左右无友军，立即按常规规则攻击一次 |
| 🌵 地刺 | 3 / 1 | **反伤**：不能主动攻击；每回合对同位置敌军反 1 血，同时自扣 1 血 |
| 😤 狂战士 | 2 / 1 | **连杀**：击杀敌人后额外攻击一次（无限连锁） |
| 🦇 吸血战士 | 3 / 1 | **吸血**：出牌献祭 1 张，攻击回血等于伤害值 |
| ✨ 祭司 | 2 / 1 | **灵活**：玩家选敌方任意位置攻击（治疗能力未实现） |
| 💀 亡灵法师 | 2 / 1 | **死亡抽牌**：场上每死 4 张牌，自己抽 1 张（攻击范围=常规近战） |
| 🐴 骑士 | 2 / 1 | **移动**：能力占位，未实现 |

### 法术 & 天气

- 🐑 **变羊术**：把敌方任意单位变成 1/1 绵羊（覆盖原有能力）
- 🌧️ **雨**：弓箭手 / 炸弹人在敌方该列前排没死时，不能越过打后排
- ☀️ **太阳**：触发时双方各 +1 抽
- ❄️ **雪**：不能优先攻击正前方，改为先打左前或右前

> 天气卡不会进入手牌——抽到时**立即触发**并自动补抽一张。

### 攻击优先级（默认）

非远程单位按以下顺序找目标：
1. **正前方**同列对面前排
2. 正前方同列对面后排
3. 左前 / 右前的列（先前后排）

被雨天 / 雪天 / 任意射 / 祭司等规则改写时按改写后规则。

## 界面操作

| 操作 | 效果 |
|---|---|
| 点击我方手牌 | 选中要出的单位 / 法术 |
| 点击我方手牌（已选中） | 取消选择 |
| 点击我方空战场格 | 放置选中单位 |
| 点击敌方战场格（战斗中 await_target 时） | 弓箭手 / 炸弹人等选目标 |
| 点击右上角 ☀️/🌙 | 切换深浅主题 |
| 「重新开始」按钮 | 重置整局 |

战斗中有目标可选时，可击目标会**高亮闪烁**。30 秒不选系统会自动帮你选第一个有效目标（防止卡死）。

## 已知规则与实现的差异

> 以下为代码实际行为，与 `规则.txt` 原文不一致的地方：

- 盗贼「孤立先手」在亮牌阶段（非战斗阶段）立即结算一次攻击
- 炸弹人「先发」**仅在该单位被部署的那一回合**战斗开始时发动一次，之后回合按弓箭手规则正常攻击
- 地刺只在**同位置有敌军**时才扣对方 + 自扣；无敌军时无动作
- 牌库用完时自动洗弃牌堆回库（Fisher-Yates，生产用 `Math.random`，测试可注入种子）
- 吸血战士出牌代价为「弃 2 张」（首回合会因代价过高无法出场）
- 亡灵法师的「每死 4 张抽 1」用**全局死亡计数**实现（双方死亡的牌都计入）
- **祭司「治疗」能力未实现**：当前仅按「任意敌方位置」攻击，未做攻击/治疗二选一
- **亡灵法师攻击范围**：实际为常规近战（同斧兵：正前后排→其他列），非「同祭司」
- **骑士「移动」能力未实现**：仅显示能力标签，攻击完不会触发换格
- **盾兵替伤**：选第一个相邻盾兵挡刀（不区分玩家偏好），可被地刺「无视替伤」绕过
- **变羊术**：变为 1/1 绵羊后仍按常规近战攻击（绵羊无特殊能力）
- **天气持续**：抽到新天气立即覆盖旧的（不会自动回合结束清除）
